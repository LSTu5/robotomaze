# RoBoToMaZe (AI Programming 1)

This repository contains the course notebook **AI_programming_1_RoBoToMaZe.ipynb**, which includes a pygame-based grid game and exercises on uninformed search (BFS/DFS).

## Setup

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

## Run

Open and run the notebook:

```bash
jupyter notebook AI_programming_1_RoBoToMaZe.ipynb
```

## Notes

- The notebook uses **pygame** and opens a game window; it may not work in headless environments.
- Recommended versions (per notebook): Python 3.10+ and pygame 2.5.2+.
